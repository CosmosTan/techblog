一、 Linux ​

根据维护团体（商业公司维护和社区组织维护）的不同，可以将众多 Linux 发行版分为 2 个系列，分别为 RedHat 系列和 Debian 系列。

  • RedHat 系列: 代表 Linux 发行版有 RedHat、CentOS、Fedora 等，使用 yum 作为包管理器；
  • Debian 系列有: Debian、Ubuntu 等，使用 apt 作为包管理器
